## `./bin/run.js tunnel`

Displays a single SSH tunnel command for all services.

```
USAGE
  $ ./bin/run.js tunnel [--network <value>]

FLAGS
  --network=<value>  [default: my-devnet] Name of the network

DESCRIPTION
  Displays a single SSH tunnel command for all services.
```
