## `./bin/run.js voting add-account`

Add a new account to Brownie in silent mode

```
USAGE
  $ ./bin/run.js voting add-account [--network <value>]

FLAGS
  --network=<value>  [default: my-devnet] Name of the network

DESCRIPTION
  Add a new account to Brownie in silent mode
```
